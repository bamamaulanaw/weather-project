# Weather Project

Weather App is a simple desktop application built with Python and PyQt5.

## Description

It allows users to enter a city name and retrieve current weather information
using the OpenWeatherMap API.

## Features

- Desktop GUI built with PyQt5
- Fetches real-time weather data
- Displays temperature, weather emoji, and description
- Handles common HTTP and network errors

## Requirements

- Python 3
- PyQt5
- requests

## Installation

```bash
pip install PyQt5 requests
```

## Usage

```bash
python main.py
```
